Evolution
Project Timeline
Three distinct phases, each building on the last while preserving the core experience.
Phase 1 — V1
Personal Tracker
A browser-only time tracker for one person. Hardcoded project list, localStorage persistence, full Amplitude instrumentation from day one.

Stack

  • React 18
  • TypeScript
  • Vite
  • LocalStorage
  • Amplitude SDK
  • GitHub Pages

Key capabilities

  • Start / Pause / Stop timers
  • Billable vs. non-billable
  • Session notes on stop
  • Daily summary
  • Google Calendar embed
  • Session Replay (100%)
Phase 2 — V2
Floating Widget (PiP)
Added an always-on-top floating timer using the Document Picture-in-Picture API. Users can track time without switching tabs.

What's new

  • Pop Out / Pop In button
  • Compact floating timer
  • Quick-start project picker
  • Notes modal in PiP
  • State sync via React Portal

New analytics

  • Widget Popped Out / In events
  • source property (main / pip_widget)
Phase 3 — Team Edition
Multi-User Platform
Replaced localStorage with Supabase. Added Google SSO, admin panel, per-user project assignments, and Row Level Security. Deployed to Vercel. Ready for 63 users.

New infrastructure

  • Supabase (Postgres)
  • Google SSO (OAuth)
  • Row Level Security
  • Admin panel
  • User profiles
  • Vercel deployment

Admin capabilities

  • Create / archive projects
  • Bulk assign (Select All / Deselect All)
  • User-centric project view
  • Manage billable categories
Phase 4 — Smart Automation (current)
Calendar Sync & Intelligence
Google Calendar integration with auto-start/stop timers, idle timeout detection, Amplitude SDK instrumentation, and self-serve project management.

What's new

  • Google Calendar API integration
  • Auto-start/stop timers from meetings
  • Smart event-to-project matching
  • Keyword memory (learns over time)
  • Idle timeout (30min + 5min warning)
  • Manage Projects with type-to-confirm removal

New analytics

  • Amplitude Browser SDK integrated
  • source: calendar for timer events
  • Calendar Event Linked
  • Idle Timeout event
  • Project Removed event
  • Sign In Completed with user identity

Capabilities
Feature Highlights
Everything the team needs to track time accurately and effortlessly.
⏱

Precision Timer

Start, pause, resume, and stop timers per project. Elapsed time updates every second. Pause tracking preserves accuracy.

↗

Floating Widget

Pop out a compact always-on-top timer using the Document PiP API. Track time without leaving your current workflow.

📝

Session Notes

Capture what you worked on when stopping a timer. Notes feed directly into automated biweekly reports.

🔒

Google SSO

Sign in with your @amplitude.com Google account. No extra passwords. Internal-only access enforced at the OAuth level.

👥

Admin Panel

Admins create projects, manage categories, and assign them to team members. Users see only their relevant projects.

📈

Amplitude Analytics

Every timer action is instrumented. Charts, funnels, and automated reports deliver insights without manual effort.

📅

Calendar Sync

Google Calendar integration auto-starts and stops timers based on your meetings. Smart matching learns which events map to which projects.

⏳

Idle Timeout

Detects 30 minutes of inactivity, warns for 5 minutes, then auto-stops and rewinds to the nearest half-hour. No more runaway timers.
